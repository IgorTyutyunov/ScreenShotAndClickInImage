В этой программе будут решены задачи по монипулированию мышкой на скриншоте экрана и при этой все действия будут отражаться на экране компьютера. 
======================================
Что сделано:
0. Работает трансляция изображение с монитора на элемент Image.
1. Работает одинарный, двойной и тройной клик ЛКП, но действия происходят при срабатывании события MouseDown, а надо чтобы был полноценный клик (MouseDown -> MouseUp).
2. Работает ПКМ, но также через собитие MouseDown.
3. Работает передвижение курсора мыши с зажатой ЛКМ, указывая в текстовых полях X1,Y1,X2,Y2 начальные и конечные координаты.
======================================

2. Реализовать перемещение курсора без зажатой кнопки мыши, а также с зажатой кнопкой мыши. Соответвенно с задержкой, типа, провили по элементу Image,
